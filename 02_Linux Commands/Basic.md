# ✅ Basic Commands

# **1️⃣ Navigation Commands**

### **`pwd`**

Shows your current working directory.

### **`whoami`**

Displays the username of the currently logged-in user.

### **`ls`**

Lists files and directories in the current folder.

### **`cd Desktop`**

Moves into the Desktop directory (relative path).

### **`cd ..`**

Moves 1 directory up.

### **`cd Documents`**

Enters the Documents directory.

### **`cd Downloads`**

Moves into the Downloads directory.

### **`cd ~`**

Takes you to your **home directory**.

---

# **2️⃣ Listing Commands**

### **`ls -lt`**

List files by **modification time (newest first)**.

### **`ls -ltr`**

List files by **modification time (oldest first)**.

### **`ls -lh`**

List with **human-readable sizes** (KB/MB/GB).

### **`ls -a`**

Shows **all files**, including hidden files (starting with `.`).

---

# **3️⃣ File Creation, Viewing & Editing**

### **`touch newfile`**

Creates an empty file named `newfile`.

### **`cat newfile`**

Displays the content of the file.

### **`less newfile`**

Views content page by page (better for long files).

### **`more newfile`**

Older version of paging through file contents.

### **`vi newfile`**

Opens the file in the **vi editor**.

### **`nano newfile`**

Opens the file in **nano editor**.

---

# **4️⃣ File & Folder Management**

### **`rm newfile`**

Deletes the file `newfile`.

### **`mkdir newfolder1`**

Creates a directory named `newfolder1`.

### **`mkdir newfolder`**

Creates another directory (“newfolder”).

### **`cp newfile newfolder`**

Copies `newfile` into the directory `newfolder`.

### **`cd newfolder`**

Moves inside the directory.

### **`cp ../new2 .`**

Copies `new2` from the parent directory (`..`) into the current directory (`.`).

### **`cp newfile n3`**

Copies `newfile` to a new file named `n3`.

### **`mv new2 newfolder`**

Moves file `new2` into directory `newfolder`.

### **`mv newfile modify`**

Renames `newfile` → `modify`.

---

# **5️⃣ Path-Related Commands**

### **`cd ../..`**

Go up **two levels** in the directory hierarchy.

### **`pwd`**

Again used to verify current path.

---

# **6️⃣ History Command**

### **`history`**

Displays previously run commands.
